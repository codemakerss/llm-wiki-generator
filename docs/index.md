## Recommended Reading Path

### If you are new here

1. Read the root [README](../README.md)
2. Check the workflow diagram
3. Pick the usage guide that matches your language

### If you want to use the project directly

1. Read [README.en-usage.md](README.en-usage.md) or [README.zh-usage.md](README.zh-usage.md)
2. Run `bootstrap-status`
3. Initialize the wiki vault
4. Start ingesting documents
